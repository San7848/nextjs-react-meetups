# React Meetups

A full-stack Next.js application to create, browse, and manage developer meetups.

## Tech Stack
- Next.js
- React
- MongoDB
- Vercel

## Features
- Create new meetups
- Dynamic routing
- SEO with Next Head
- MongoDB integration
- Server-side rendering
